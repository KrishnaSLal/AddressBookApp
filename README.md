# ✅ UC12 - Sort Contacts by Name

## ✅ Use Case
Ability to sort the entries in the Address Book alphabetically by person's name.

## 📌 Objective
Display contacts in alphabetical order based on person name.

## 🛠 Implementation Details
- Used Collection Library for sorting
- Used Console to trigger sorting
- Overrode `toString()` method for display
- Used Java Streams where required

## 🎯 Outcome
Contacts are displayed in sorted order by name.
