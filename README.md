# ✅ UC15 - Read/Write Address Book as CSV File

## ✅ Use Case
Ability to read/write the Address Book with persons' contact as CSV file.

## 📌 Objective
Store and retrieve Address Book data in CSV format.

## 🛠 Implementation Details
- Used OpenCSV library
- Converted Address Book entries into CSV format
- Read CSV file data back into application objects

## 🎯 Outcome
Contacts can be persisted and loaded using CSV files.
