# ✅ UC8 - Search Person by City or State

## ✅ Use Case
Ability to search person in a city or state across multiple Address Books.

## 📌 Objective
Find all contacts belonging to a given city or state.

## 🛠 Implementation Details
- Searched across multiple Address Books
- Search result can contain multiple persons
- Used Java Streams for filtering contacts

## 🎯 Outcome
