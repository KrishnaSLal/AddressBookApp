# UC6 - Add Multiple Address Books

## ✅ Use Case
Refactor to add multiple Address Books to the system, each with a unique name.

## 📌 Objective
Support more than one Address Book in the system.

## 🛠 Implementation Details
- Used Console to add new Address Books
- Maintained Dictionary/Map of AddressBook name to AddressBook object
- Ensured each Address Book has a unique name

## 🎯 Outcome
System can manage multiple Address Books independently.
