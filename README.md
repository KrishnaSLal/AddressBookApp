# ✅ UC22 - Read Entries from JSON Server

## ✅ Use Case
Ability to read entries of Address Book from JSON Server.

## 📌 Objective
Fetch Address Book data from JSON Server and update in-memory data.

## 🛠 Implementation Details
- Used REST Assured for REST API calls from JUnit test code
- Retrieved contact entries from JSON Server
- Updated Address Book application memory with fetched data

## 🎯 Outcome
Address Book entries can be read from JSON Server and synced with application memory.
