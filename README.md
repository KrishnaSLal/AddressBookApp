# ✅ UC7 - Prevent Duplicate Entries

## ✅ Use Case
Ability to ensure there is no duplicate entry of the same person in a particular Address Book.

## 📌 Objective
Avoid adding the same contact more than once in the same Address Book.

## 🛠 Implementation Details
- Duplicate check performed using person name
- Used Collection methods to search contact before adding
- Overrode `equals()` method for duplicate comparison
- Used Java Streams for searching

## 🎯 Outcome
Duplicate contacts are prevented within an Address Book.
