# ✅ UC23 - Add Multiple Entries to JSON Server

## ✅ Use Case
Ability to add multiple entries to Address Book JSON Server and sync with Address Book application memory.

## 📌 Objective
Insert multiple contacts into JSON Server and keep application memory updated.

## 🛠 Implementation Details
- Used REST Assured for REST API calls from JUnit test code
- Added multiple contact records to JSON Server
- Synced inserted records with in-memory Address Book data

## 🎯 Outcome
Multiple contacts can be added to JSON Server successfully and synchronized with the application.
