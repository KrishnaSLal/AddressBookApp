# ✅ UC9 - View Persons by City or State

## ✅ Use Case
Ability to view persons by city or state.

## 📌 Objective
Group and display contacts based on city or state.

## 🛠 Implementation Details
- Maintained Dictionary/Map of city to persons
- Maintained Dictionary/Map of state to persons
- Used Collection Library and Java Streams

## 🎯 Outcome
Contacts can be viewed city-wise and state-wise.
