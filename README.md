# ✅ UC25 - Delete Entry from JSON Server

## ✅ Use Case
Ability to delete entry in Address Book JSON Server and sync with Address Book application memory.

## 📌 Objective
Remove a contact from JSON Server and reflect deletion in memory.

## 🛠 Implementation Details
- Used REST Assured for REST API calls from JUnit test code
- Deleted the target contact from JSON Server
- Updated Address Book application memory after deletion

## 🎯 Outcome
Contacts can be deleted from JSON Server with proper synchronization in application memory.
