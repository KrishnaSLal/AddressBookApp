# ✅ UC14 - Read/Write Address Book Using File IO

## ✅ Use Case
Ability to read or write the Address Book with persons' contact into a file using File IO.

## 📌 Objective
Persist Address Book data into a file and retrieve it when needed.

## 🛠 Implementation Details
- Used Java File IO
- Wrote Address Book data into file
- Read file content back into the application

## 🎯 Outcome
Address Book data can be stored and retrieved through file handling.
