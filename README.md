# ✅ UC4 - Delete a Contact

## ✅ Use Case
Ability to delete a person using the person's name.

## 📌 Objective
Remove a contact from the Address Book by searching with name.

## 🛠 Implementation Details
- Used Console input to accept person name
- Searched contact in Address Book
- Deleted the matching contact record

## 🎯 Outcome
A contact can be removed successfully from the Address Book.
