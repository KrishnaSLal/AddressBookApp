# ✅ UC21 - Add Multiple Contacts to Database Using Threads

## ✅ Use Case
Ability to add multiple contacts to the Address Book database.

## 📌 Objective
Improve insertion performance by adding multiple contacts concurrently.

## 🛠 Implementation Details
- Used JDBC for database operations
- Used transactions where multiple tables were impacted
- Used Threads for multiple DB insertions
- Ensured parallel handling of multiple contact entries

## 🎯 Outcome
Multiple contacts can be inserted into the database efficiently using multithreading.
