# ✅ UC18 - Retrieve Contacts Added in a Particular Period

## ✅ Use Case
Ability to retrieve contacts from the database that were added in a particular period.

## 📌 Objective
Fetch records between specific date ranges.

## 🛠 Implementation Details
- Used JDBC for CRUD operations
- Introduced `date_added` field in the contacts table
- Queried records based on time period

## 🎯 Outcome
Contacts added within a selected date range can be retrieved from the database.
