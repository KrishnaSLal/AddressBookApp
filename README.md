# ✅ UC10 - Count Contacts by City or State

## ✅ Use Case
Ability to get number of contact persons by city or state.

## 📌 Objective
Count how many contacts belong to a specific city or state.

## 🛠 Implementation Details
- Used Java Streams for counting
- Search result displays total count by city
- Search result displays total count by state

## 🎯 Outcome
Users can get contact counts grouped by city or state.
