# ✅ UC19 - Count Contacts by City or State from Database

## ✅ Use Case
Ability to retrieve number of contacts in the database by city or state.

## 📌 Objective
Generate city-wise and state-wise contact counts directly from DB.

## 🛠 Implementation Details
- Used JDBC for database operations
- Used database functions for aggregation
- Retrieved grouped count results by city/state

## 🎯 Outcome
Contact counts by city or state can be fetched from the database.
