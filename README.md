# ✅ UC24 - Update Entry in JSON Server

## ✅ Use Case
Ability to update entry in Address Book JSON Server and sync with Address Book application memory.

## 📌 Objective
Modify existing contact data in JSON Server and reflect the changes in memory.

## 🛠 Implementation Details
- Used REST Assured for REST API calls from JUnit test code
- Updated contact details on JSON Server
- Synced updated records with Address Book application memory

## 🎯 Outcome
Updated JSON Server records remain consistent with application memory.
