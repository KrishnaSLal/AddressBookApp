# ✅ UC13 - Sort Contacts by City, State, or Zip

## ✅ Use Case

## 📌 Objective
Provide flexible sorting options for Address Book entries.

## 🛠 Implementation Details
- Implemented functions to sort by city
- Implemented functions to sort by state
- Implemented functions to sort by zip
- Used Collection Library and Java Streams

## 🎯 Outcome
Contacts can be sorted using different location-based fields.
