# UC5 - Add Multiple Contacts

## ✅ Use Case
Ability to add multiple persons to the Address Book.

## 📌 Objective
Store more than one contact in the Address Book.

## 🛠 Implementation Details
- Used Console to add person details one at a time
- Used Collection classes to manage multiple contacts
- Maintained list of contact persons in Address Book

## 🎯 Outcome
Multiple contacts can be added and managed in the Address Book.
